# Face Detection Program (Python + OpenCV)

This project detects faces using your webcam in real-time using OpenCV Haarcascade.

## Features
- Real-time face detection
- Green rectangle around face
- Works with webcam
- Press Q to quit

## Installation
Install OpenCV:

pip install opencv-python

## How to Run
python face_detection.py

## Model File
Make sure this file exists:
models/haarcascade_frontalface_default.xml
